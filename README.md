# 🖼️ 图像处理与颜色识别工具

> 一个基于 OpenCV 的多功能图像处理与实时颜色检测工具，支持图像基础操作、滤波、形态学变换、轮廓分析及摄像头实时颜色识别。


---



## 🔧 功能概览

本项目包含以下主要功能模块：

### 1. 图像基础操作
- 读取与显示图像
- 查看图像属性（形状、数据类型、像素总数）
- 图像保存（支持 PNG 格式）

### 2. 图像处理流程（快捷键 `k`）
一键执行：
- 裁剪为正方形区域（从指定位置裁剪）
- 缩放至 300×300
- 水平翻转
- 自动保存处理结果

### 3. 边缘检测与轮廓分析（快捷键 `e`）
- 灰度化、二值化
- Canny 边缘检测
- 轮廓提取与绘制
- 外接矩形标注
- 面积与周长计算并可视化显示

### 4. 图像滤波对比（快捷键 `f`）
支持多种滤波算法对比：
- 均值滤波
- 高斯滤波
- 中值滤波（有效去除椒盐噪声）
- 双边滤波（保边去噪）
- 自动生成滤波效果对比图

### 5. 形态学操作（快捷键 `m`）
- 腐蚀、膨胀
- 开运算（去噪点）、闭运算（补洞）
- 梯度运算（边缘提取）
- 支持矩形、椭圆、十字结构元
- 自适应阈值分割（应对光照不均）

### 6. 颜色空间分析（HSV）
- 将图像转换为 HSV 色彩空间
- 提取指定颜色区域（示例：蓝色）
- 显示 H（色相）、S（饱和度）、V（明度）各通道
- 掩码生成与颜色分离结果展示

### 7. 实时颜色检测（摄像头）
- 调用电脑摄像头实时捕获画面
- 检测指定颜色物体（红、蓝、绿、黄）
- 绘制边界框与标签
- 显示检测数量统计
- 按 `'q'` 退出程序



---



## 🛠️ 技术栈

- **Python** 3.7+
- **OpenCV** (`cv2`)：图像处理核心库
- **NumPy**：数组运算支持
- **HSV 颜色空间**：对光照变化鲁棒性强
